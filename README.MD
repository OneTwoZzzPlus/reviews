# Для желающих разрабатывать...

### Сборка сайта (`build-page.js`) и расширения (`build.js`)

Установка
```shell
npm init -y
npm install --save-dev esbuild
npm install --save-dev cross-env
npm install --save-dev archiver
```

Сборка сайта
```shell
node run build:prod
# или если api работает на localhost
node run build:dev
```

Сборка расширения
```shell
node run build-page:prod
# или если api работает на localhost
node run build-page:dev
```

### Деплой сайта (Github Pages)
```shell
node run build-page:prod
git add .
git commit -m "commit_message"
git push -u origin main 
```

### Релиз расширения `release.js`
```shell
node run release
```
- Версию нужно указывать в `manifest.json`.
- Для описания релиза будет использовано `release-notes.md`.
- Может сойти за глупенький ci/cd.

### Содержимое `src`

Изначально у нас три путя
1. `page.html` запускает `page.js` который обеспечивает работу сайта
2. `popup.html` запускает `popup.js` который обеспечивает работу окошечка
3. `injector.js` производит встраивание на сайт блока отзывов и поиск jwt токена

В `manifest.json` описаны основные характеристики браузерного расширения, необходимые браузеру.

В `style.css` хранятся топовые стили. **Ни в коем случае** не переназначайте общие стили элементов тут, используйте для этого `popup.html`, иначе переопределяться стили сайта.

В остальном разберётесь по ходу дела)
